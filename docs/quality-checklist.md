## Техническая проверка

- [x] Незаполненные маркеры и демонстрационные строки удалены.
- [x] Все относительные Markdown-ссылки разрешаются.
- [x] Внешние URL проверены с переходом по перенаправлениям; доступность
  отдельных сервисов может зависеть от региона.
- [x] В репозитории нет `.DS_Store`, временных файлов офисных приложений и
  пустых либо незаполненных шаблонных каталогов.
- [x] Все отслеживаемые файлы значительно меньше ограничения GitHub в 100 МБ;
  крупнейший файл — `data/krm-v3.0.xlsx`, менее 0,5 МБ.
- [x] `git diff --check` не обнаруживает ошибок пробельного оформления.

## Примечание об исходных материалах

Файлы DOCX, XLSX, PDF и TeX в каталоге [`data`](../data/README.md) являются
штатными исходными материалами курса. Они не считаются временными файлами;
публикуемая Markdown-версия рабочей программы находится в
[`docs/rpd.md`](rpd.md).
