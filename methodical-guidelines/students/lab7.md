# Лабораторная работа №7: Файловый ввод-вывод: чтение CSV

## Тема 

**Чтение и парсинг структурированных текстовых файлов.**

## Задачи

1. Подготовить CSV-файл books.csv (50+ строк) с полями: id,title,author,year,isbn,publisher.
2. Написать парсер CSV:
Чтение через BufferedReader построчно.
Разбиение строки с учётом кавычек (поля, содержащие запятые внутри кавычек).
Создание объектов Book из каждой строки.
3. Обработать некорректные строки: недостаточно полей, нечисловой год.
4. Вывести статистику: всего строк, успешно прочитано, пропущено с ошибками, дубликатов по ISBN.
5. Реализовать то же самое через Files.readAllLines() + Stream API — сравнить читаемость кода.
6. Записать результат (успешно прочитанные книги) в новый CSV-файл books_clean.csv.
7. Сгенерировать регулярное выражение для разбиения CSV-строки с учётом кавычек, обсудить его корректность на тестовых примерах.


## Стек технологий

- **Язык:** Java (версия 17+)
- **IDE:** JetBrains IntelliJ IDEA Community Edition
- **VCS:** Git
- **ИИ-помощник:** GitHub Copilot (или аналоги)

## Инструкция по выполнению

### 1. Подготовка

```bash

```

Подготовить файл src/main/resources/books.csv (минимум 50 строк). Обязательно включить строки с запятыми внутри кавычек, например:
3,"Мастер и Маргарита","М.А. Булгаков",1967,"978-5-17-12345","Издательство АСТ"

### 2. Парсер через BufferedReader (Low-level I/O)

- Открыть поток чтения (Files.newBufferedReader) в блоке try-with-resources.
- Прочитать первую строку (хедер).
- В цикле читать остальные строки. Разделять их на токены, учитывая экранирование кавычками (см. пункт про ИИ).
- При ошибке парсинга (например, год — не число) бросать или логировать исключение, не прерывая чтение всего файла.

### 3. Обработка ошибок и дубликатов

- Использовать Set<String> isbnSet для отслеживания уникальных ISBN.
- Считать количество пропущенных строк из-за дублей по ISBN.
- Собирать успешно созданные объекты Book в список.

### 4. Статистика

- Вывести в консоль итоговые цифры: [Всего строк] / [Успешно] / [Ошибки формата] / [Дубликаты].

### 5. Stream API (High-level I/O)

- Реализовать ту же логику чтения одной строкой: Files.lines(path).skip(1)...collect(Collectors.toList()).
- Сравнить объем кода и читаемость двух подходов.

### 6. Запись в файл (Writing)

- Записать очищенный список книг в books_clean.csv.
Если название содержит запятую — обернуть его в кавычки при записи.

### 7. Генерация через ИИ (Copilot)

Протестировать предложенное регулярное выражение на строке с вложенными кавычками.

## Варианты

1. Банковское дело (BankAccount):
Чтение: accounts.csv (id,owner,balance,currency).
Ошибка: Пропуск строк с отрицательным балансом.
Статистика: Сумма всех активов в базовой валюте.
2. Автомобильный транспорт (Car):
Чтение: cars.csv (vin,model,year,color).
Ошибка: Неверная длина VIN (не 17 символов).
Статистика: Количество машин старше 15 лет.
3. Электронная коммерция (Product):
Чтение: products.csv (sku,name,price,stock).
Ошибка: Отрицательное количество на складе.
Статистика: Общая стоимость всего складского запаса.
4. Социальные сети (UserProfile):
Чтение: users.csv (uid,login,email,date_joined).
Ошибка: Некорректный формат email (нет символа @).
Статистика: Регистрации по месяцам (используя Map<YearMonth, Long>).
5. Геолокация (GeoPoint):
Чтение: route.csv (lat,lon,timestamp).
Ошибка: Значения широты вне диапазона [-90, 90].
Статистика: Общее расстояние маршрута (сумма отрезков).
6. Университет (Student):
Чтение: grades.csv (student_id,course,grade).
Ошибка: Оценка не из множества {A, B, C, D, F}.
Статистика: Средний балл (GPA) каждого студента.
7. Видеоигры (VideoGame):
Чтение: library.csv (title,platform,size_gb).
Ошибка: Несуществующая платформа (проверка через Enum).
Статистика: Общий объем установленных игр.
8. Медицина (PatientCard):

Статистика: Группировка пациентов по году рождения.
9. Библиотечное дело (LibraryItem):
Чтение: catalog.csv (barcode,title,author,isbn).
Ошибка: Пустой ISBN или Barcode.
Статистика: Топ-5 самых частых авторов.
10. Космонавтика (Spacecraft):
Чтение: fleet.csv (name,type,status).

Статистика: Количество кораблей в статусе "Active".
11. Умный дом (SmartDevice):

Ошибка: Некорректный формат MAC-адреса.
Статистика: Устройства с батареей ниже 15%.
12. Ресторанный бизнес (MenuItem):
Чтение: menu.csv (id,name,category,price,allergens).

Статистика: Средняя цена блюд по категориям.

Чтение: playlist.csv (artist,duration_sec,explicit).
Ошибка: Длительность трека < 1 секунды.
Статистика: Общая длительность плейлиста в формате MM:SS.
