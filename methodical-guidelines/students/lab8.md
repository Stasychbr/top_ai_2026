# Лабораторная работа №8: Файловый ввод-вывод: JSON и XML

## Тема

**Сериализация и десериализация структурированных форматов. Библиотека Jackson.**

## Задачи

1. Подключить Jackson (зависимость Maven/Gradle). Разобрать аннотации: @JsonProperty, @JsonIgnore, @JsonFormat.
2. Прочитать файл books.json, содержащий массив объектов Book в JSON-формате, в List<Book>.
3. Сериализовать список книг обратно в JSON и записать в файл.
4. Реализовать конвертер: на вход — books.csv, на выход — books.json (первый простой ETL-пайплайн).
5. Реализовать чтение того же набора данных из XML с использованием Jackson XmlMapper.
6. Добавить аннотации для форматирования дат (@JsonFormat(pattern = "yyyy-MM-dd")) на примере поля addedDate.
7. генерировать POJO-класс по образцу JSON через Copilot, проверить корректность типов данных.

## Стек технологий

- **Язык:** Java (версия 17+)
- **IDE:** JetBrains IntelliJ IDEA Community Edition

- **ИИ-помощник:** GitHub Copilot (или аналоги)

## Варианты

Общая часть для всех заданий: 
- Вход: CSV-файл из предыдущего задания.
- Процесс: Прочитать данные, провалидировать их и создать список объектов (List<T>).
- Выход 1: JSON-файл с результатами (output.json).
- Выход 2: XML-файл с теми же данными (output.xml) через XmlMapper.
- Аннотации: Обязательно использовать @JsonProperty, @JsonIgnore (для внутренних ID) и @JsonFormat (для дат/времени).
