## Общие правила

- Номер варианта назначает преподаватель. Один номер и соответствующая предметная область используются во всех работах.
- Технические требования КИМ обязательны. Сущности, поля и предметные операции берутся из файла вариантов, на который ссылается КИМ.
- ИИ-агент используется только в границах, прямо указанных в задании. Студент сохраняет промпты, проверяет результат и самостоятельно отвечает за окончательный код.
- В ИИ-агент нельзя передавать пароли, токены, персональные данные и закрытые материалы.
- Проект должен собираться и запускаться в воспроизводимом окружении; все представленные тесты необходимо выполнить до сдачи.
