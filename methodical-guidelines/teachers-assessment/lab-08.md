# Рубрика лабораторной работы №8

## Проверяемые результаты

- Jackson подключён воспроизводимо, модель типизирована и аннотации применены по назначению.
- JSON читается и записывается без потери обязательных данных.
- Конвертер CSV → JSON использует правила обработки ошибок предыдущей работы.
- XML записывается самостоятельно и читается единственным методом, разрешённым к генерации.
- Проверены даты, специальные символы, пустые и неизвестные поля.
- Спецификация метода была составлена до обращения к агенту; результат проверен и исправлен.

## Существенные недостатки

- предметная модель заменена необобщёнными `Map`;
- циклическое преобразование теряет или искажает данные;
- агенту поручена вся лабораторная либо JSON-часть;
- сгенерированный метод принят без тестирования повреждённого XML.
