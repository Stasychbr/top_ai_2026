# Лабораторная работа №1: Настройка окружения. Первый класс

## Тема

**Установка JDK, IntelliJ IDEA (Community Edition), Git. Создание класса с инкапсуляцией.**

## Задачи

1. Установить среду выполнения JDK 17 или выше и интегрированную среду разработки IntelliJ IDEA Community Edition.
2. Инициализировать пустой Git-репозиторий для проекта, настроить .gitignore для игнорирования служебных файлов IDE и скомпилированного кода (*.class, /out/), выполнить первый коммит.
3. Создать проект в IntelliJ IDEA и для каждой из указанных областей необходимо создать класс на языке Java, соблюдая принципы инкапсуляции:

    - все поля объявлены как private.
    - для каждого поля реализованы публичные методы доступа (getter) и изменения (setter).
    - реализовать метод getDescription(), формирующий строковое представление объекта по заданному шаблону.

4. Написать код в методе main() класса для инстанцирования трех различных объектов класса и вывода результата их метода getDescription() в консоль.
5. Протестировать работу программы, зафиксировать промежуточное состояние через коммиты в Git.
6. Сгенерировать шаблон класса с помощью ИИ-инструмента (GitHub Copilot) по текстовому комментарию, сравнить полученный результат с ручным написанием по критериям читаемости, полноты методов и соответствия стандартам оформления кода.

## Варианты

1. Банковское дело (BankAccount)
Реализовать класс банковского счета с полями: номер счета, ФИО владельца, баланс, валюта.
2. Автомобильный транспорт (Car)
Реализовать класс автомобиля с полями: марка, модель, VIN-номер, пробег, год выпуска.
3. Электронная коммерция (Product)
Реализовать класс товара с полями: артикул (SKU), наименование, категория, цена, количество на складе.
4. Социальные сети (UserProfile)
Реализовать профиль пользователя с полями: уникальный никнейм, полное имя, биография, аватар (URL), статус приватности.
5. Геолокация (GeoPoint)
Реализовать точку на карте с полями: широта, долгота, высота над уровнем моря.
6. Университет (Student)
Реализовать карточку студента с полями: ID зачетки, имя, фамилия, факультет, текущий средний балл (GPA).
7. Видеоигры (VideoGame)
Реализовать сущность игры с полями: название, разработчик, жанр, год релиза, платформа.
8. Медицина (PatientCard)
Реализовать медицинскую карту с полями: ID карты, ФИО пациента, дата рождения, основной диагноз.
9. Библиотечное дело (LibraryItem)
Реализовать библиотечную единицу с полями: инвентарный номер, заглавие, автор, шифр полки, статус выдачи.
10. Космонавтика (Spacecraft)
Реализовать космический аппарат с полями: обозначение/название, год запуска, тип миссии, вместимость экипажа.
11. Умный дом (SmartDevice)
Реализовать устройство умного дома с полями: идентификатор устройства, тип (лампа, розетка), версия прошивки, статус онлайн.
12. Ресторанный бизнес (MenuItem)
Реализовать позицию меню с полями: ID блюда, название, состав, вес порции, цена.
13. Музыка (MusicTrack)
Реализовать музыкальный трек с полями: ID трека, название, исполнитель, длительность (в секундах), альбом.
14. Управление проектами (Task)
Реализовать задачу проекта с полями: ID задачи, краткое описание, ответственный исполнитель, статус (To Do, In Progress, Done), дедлайн.
15. Зоопарк (Animal)
Реализовать карточку животного с полями: чип-ID, вид, кличка, номер вольера, возраст.
16. Фитнес (Exercise)
Реализовать упражнение с полями: ID упражнения, название, целевая группа мышц, количество подходов, количество повторений.
17. Кибербезопасность (NetworkCredential)
Реализовать учетные данные с полями: логин, хеш пароля, включен ли двухфакторный вход, дата создания записи.
18. Сельское хозяйство (CropField)
Реализовать сельскохозяйственное угодье с полями: кадастровый номер участка, культура, площадь (гектары), прогнозируемая урожайность.
19. Event-менеджмент (ConcertTicket)
Реализовать билет на концерт с полями: ID билета, артист, площадка, дата и время, ряд, место.
20. Экология (RecyclableWaste)
Реализовать запись об отходах с полями: ID партии, тип материала (пластик, стекло, бумага), масса (кг), пункт приема.
