# Лабораторная работа №8: Файловый ввод-вывод: JSON и XML


## Варианты

Общая часть для всех заданий: 
- Вход: CSV-файл из предыдущего задания.
- Процесс: Прочитать данные, провалидировать их и создать список объектов (List<T>).
- Выход 1: JSON-файл с результатами (output.json).
- Выход 2: XML-файл с теми же данными (output.xml) через XmlMapper.
- Аннотации: Обязательно использовать @JsonProperty, @JsonIgnore (для внутренних ID) и @JsonFormat (для дат/времени).

Варианты заданий

1. Банковское дело (BankAccount)
Валюта: Конвертировать баланс в базовую валюту (например, RUB) по фиксированному курсу перед сериализацией.
Статистика в файле: Добавить в конец JSON объект {"totalAssets": 12345.67}.

2. Автомобильный транспорт (Car)
Скрытие данных: Аннотацией скрыть полный VIN, оставив только последние 4 символа при записи в JSON.
Группировка: В JSON вывести массив объектов { "brand": "Toyota", "count": 5 }.
3. Электронная коммерция (Product)
Форматирование цены: Использовать @JsonSerialize или кастомный метод для вывода цены как строки "1 999.00 ₽".
Статус склада: Поле stockStatus ("In Stock", "Low Stock"), которого нет в CSV, вычислять на лету.
4. Социальные сети (UserProfile)
Домен почты: Извлечь домен из email и добавить как отдельное поле emailDomain в JSON.
Фильтрация: Не записывать в файл пользователей без заполненного поля date_joined.
5. Геолокация (GeoPoint)
Формат даты: timestamp из CSV преобразовать в ISO-строку (2023-10-05T14:30:00Z).
Расстояние: Добавить финальную точку маршрута и общее расстояние в корневой объект JSON.
6. Университет (Student)
Конвертер оценки: Написать конвертер, который превращает букву "A" в число 4.0 внутри объекта GPA.
Рейтинг: Вывести отсортированный список студентов по убыванию среднего балла.
7. Видеоигры (VideoGame)
Enum: Платформа должна быть Enum'ом (PC, PS5), а не строкой.
Размер диска: Вычислить суммарный объем и записать его в корень JSON как totalStorageUsedGb.
8. Медицина (PatientCard)
Возраст: Рассчитать возраст пациента от birthdate до текущей даты и добавить поле age.
Анонимизация: Скрыть СНИЛС полностью, заменив его на хэш-код.
9. Библиотечное дело (LibraryItem)
Агрегация авторов: Вместо списка книг вывести структуру:
json
Копировать
[
  { "author": "Толстой", "books": ["Война и мир", "Анна Каренина"] }
]
10. Космонавтика (Spacecraft)
Локализация статуса: Статус "Active" превратить в локализованную строку "В полете".
Тип корабля: Разделить вывод на два массива: orbitalStations и landers.
11. Умный дом (SmartDevice)
 MAC-адрес: Преобразовать MAC в верхний регистр и вставить двоеточия (AA:BB:CC...), если их нет.
Критичные устройства: Отдельный массив criticalDevices для тех, у кого батарея <15%<15%.
12. Ресторанный бизнес (MenuItem)
Аллергены: Массив аллергенов [ "Gluten", "Nuts" ] вместо строки.
Категории: Группировка блюд по категориям (Напитки, Горячее) прямо в структуре JSON.
13. Музыка (MusicTrack)
Длительность: Перевести секунды в формат mm:ss (например, 245 →"04:05").
Явный контент: Флаг explicit перевести на русский (true→ "Да").
14. Управление проектами (Task)
Прогресс: Если статус "Done", добавлять поле progress: 100. Иначе считать прогресс исходя из общего объема спринта.
Блокеры: Список заблокированных задач вынести отдельно.
15. Зоопарк (Animal)
Рацион: Расшифровать код рациона (CARN, HERB) в полное название.
Энергозатраты: Вычислить примерную потребность в калориях и добавить в JSON.
16. Фитнес (Exercise)
Тоннаж: Для каждой строки вычислить sets * reps * weight и записать в поле totalVolume.
Итог тренировки: Суммарный тоннаж за всю тренировку в конце файла.
17. Кибербезопасность (NetworkCredential)
География: Определить страну по IP (используя простую заглушку/мэп) и добавить поле country.
Опасные действия: Выделить все события типа "BruteForce" в отдельный лог-файл security_alerts.json.
18. Сельское хозяйство (CropField
Урожай: Умножить площадь на урожайность и записать результат projectedYield в тоннах.
Тип культуры: Группировка полей по типу растения.
19. Event-менеджмент (ConcertTicket)
Цена VIP: Применить коэффициент 1.5 к цене билетов сектора "VIP" при сериализации.
Карта зала: Сгруппировать билеты по рядам: { "row": "A", "seats": [1, 2, 3] }.
20. Экология (RecyclableWaste)
Категория материала: По коду пластика определить тип упаковки (Бутылка, Пленка).
Эко-рейтинг: Посчитать средний балл на килограмм отходов.
