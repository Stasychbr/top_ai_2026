# Лабораторная 13. ETL-пайплайн: Extract + Transform (данные из файлов в БД)

## Варианты

Общая задача: Написать универсальный код чтения (Extract) из CSV/JSON и сложную бизнес-логику очистки/доставки данных (Transform).

1. Банковское дело (BankAccount)
Extract: Чтение transactions.csv.
Transform: Группировка транзакций по месяцу, расчет итогового баланса за период, фильтрация ошибочных платежей (сумма = 0).
2. Автомобильный транспорт (Car)
Extract: Чтение парка машин из JSON-файла дилера.
Transform: Нормализация VIN (удаление пробелов), перевод названий цветов с английского на русский через справочник (Map), обогащение данными о ТО из второго файла.
3. Электронная коммерция (Product)
Extract: Парсинг прайс-листа поставщика в формате CSV (cp1251 encoding).
Transform: Конвертация валюты (USD → RUB) по курсу из константы, округление цен до 99 руб., фильтрация товаров не в наличии.
4. Социальные сети (UserProfile)
Extract: Импорт контактов из адресной книги (CSV).
Transform: Разделение поля "Full Name" на firstName и lastName, дедупликация пользователей по email (оставить самый свежий профиль).
5. Геолокация (GeoPoint)
Extract: Чтение сырых GPS-треков (формат NMEA или простой CSV).
Transform: Фильтрация точек со скоростью <2 км/ч (остановки), вычисление общего расстояния трека (Haversine formula внутри Stream).
6. Университет (Student)
Extract: Загрузка списка абитуриентов из Excel (через Apache POI).
Transform: Проверка наличия обязательных предметов ЕГЭ, расчет среднего балла аттестата, сортировка по рейтингу.
7. Видеоигры (VideoGame)
Extract: Чтение логов достижений игроков (JSON Lines).
Transform: Агрегация: подсчет количества платиновых трофеев у каждого игрока, поиск самого редкого достижения.
8. Медицина (PatientCard)
Extract: Выгрузка реестра пациентов из старой системы (XML).
Transform: Маскирование СНИЛС (видны только последние 4 цифры), нормализация телефонных номеров к формату +7....
9. Библиотечное дело (LibraryItem)
Extract: Каталог книг от издательства (CSV).
Transform: Генерация короткого описания (аннотации) на основе первых 200 символов полного текста, извлечение года издания из строки копирайта "(c) 2023".
10. Космонавтика (Spacecraft)
Extract: Пакет телеметрии бортового компьютера (бинарный файл или сложный текстовый дамп).
Transform: Преобразование HEX-кодов датчиков в физические величины (температура, давление), фильтрация шума.
11. Умный дом (SmartDevice)
Extract: Экспорт настроек умного дома (YAML/JSON).
Transform: Валидация IP-адресов устройств, группировка девайсов по комнатам, замена устаревших имен протоколов.
12. Ресторанный бизнес (MenuItem)
Extract: Меню в PDF (текстовое). (Сложный вариант) или CSV.
Transform: Расчет калорийности БЖУ блюда на основе веса ингредиентов, выделение вегетарианских позиций.
13. Музыка (MusicTrack)
Extract: Метаданные MP3-библиотеки (ID3 теги считываются как Map).
Transform: Приведение жанров к единому стандарту ("Rock'n'Roll" → "Rock"), заполнение пустых полей "Artist" значением "Unknown".
14. Управление проектами (Task)
Extract: Экспорт задач из Jira (CSV).
Transform: Построение дерева зависимостей (Parent-Child), расчет времени выполнения задачи (разница дат).
15. Зоопарк (Animal)
Extract: Книга учета кормов (Excel).
Transform: Перевод граммов рациона в килограммы, проверка нормы кормления (если вес животного известен, проверить % от массы тела).
16. Фитнес (Exercise)
Extract: Дамп тренировок из приложения (JSON).
Transform: Суммирование подходов (sets) если они разбиты строками, конвертация фунтов (lbs) в килограммы.
17. Кибербезопасность (NetworkCredential)
Extract: Лог авторизации веб-сервера (Apache/Nginx access.log).
Transform: Парсинг строки лога регулярными выражениями, поиск неудачных попыток входа (Status 401), группировка по IP.
18. Сельское хозяйство (CropField)
Extract: Данные спутникового мониторинга NDVI (CSV с координатами).
Transform: Классификация участков по здоровью растений (цветовая шкала), расчет площади проблемных зон.
19. Event-менеджмент (ConcertTicket)
Extract: Продажи билетов от билетного оператора.
Transform: Группировка мест согласно схеме зала (сектор, ряд), применение промокодов и скидок из справочника.
20. Экология (RecyclableWaste)
Extract: Весовые сводки с полигонов (CSV).
Transform: Расчет процента переработки от общей массы, классификация типа мусора по коду ФККО (справочник).
