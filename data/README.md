# Исходные материалы

Каталог содержит исходные документы, использованные при подготовке курса.
Файлы DOCX, XLSX, PDF и TeX являются штатной частью репозитория, а не
временными файлами офисных приложений. Актуальная публикуемая версия рабочей
программы находится в [`docs/rpd.md`](../docs/rpd.md).

## КРМ 3.0

Книга `krm-v3.0.xlsx` содержит листы с матрицей ролей, определениями,
компетенциями и базовыми дескрипторами. При переносе в документы дисциплины
исходные коды и индикаторы сохраняются, а предметные дескрипторы формулируются
отдельно.
