# Лабораторные работы

Первые 15 индивидуальных работ используют одну назначенную предметную область. Модель постепенно получает коллекционные операции, файловые представления, конкурентную обработку, хранение в PostgreSQL и ETL-пайплайн. Командный мини-проект №16 переносит эти навыки в единый серверный сценарий обработки прайс-листов.

## Как читать материалы

1. Найдите работу в таблице ниже и откройте README её модуля.
2. Изучите КИМ и требования своего варианта.
3. До начала работы прочитайте рубрику, [модель уровней КРМ](../methodical-guidelines/teachers-assessment/krm-level-model.md) и связанную инструкцию по промптам.
4. Выполняйте маркированный порядок: самостоятельное действие → ограниченное действие с ИИ → проверка результата.
5. Сдайте код и все перечисленные артефакты, включая точные промпты, существенные ответы и журнал исправлений.

Обязательная часть каждой работы измеряет Б. Отдельные задания С/П находятся в работах №13–16 и выполняются только по предварительному назначению преподавателя; их наличие не меняет обязательный сценарий.
