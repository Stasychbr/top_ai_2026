# Модуль 6. ETL-пайплайн

Модуль объединяет файловые источники, преобразование данных, JDBC-загрузку, идемпотентность, метрики и обработку ошибок.

| Лабораторная работа | Элемент РПД | Индикаторы | Условие | Варианты | Критерии |
|---|---|---|---|---|---|
| №13. Extract + Transform | 6.2 | PL-2.2, LLM-5.1, LLM-5.2 | [КИМ](kim-01-etl-extract-transform.md) | [20 вариантов](../../methodical-guidelines/teachers-resources/lab13_V.md) | [Рубрика](../../methodical-guidelines/teachers-assessment/lab-13.md) |
| №14. Load + мониторинг | 6.3 | PL-2.2, LLM-5.1, LLM-5.2 | [КИМ](kim-02-etl-load-monitoring.md) | [20 вариантов](../../methodical-guidelines/teachers-resources/lab14_V.md) | [Рубрика](../../methodical-guidelines/teachers-assessment/lab-14.md) |

Обязательные части измеряют Б. Связанная пара C13-PL2.2 + C14-PL2.2 измеряет С по PL-2.2; правила накопления и доказательства приведены в [модели уровней КРМ](../../methodical-guidelines/teachers-assessment/krm-level-model.md).

Во всех работах используется один номер варианта, назначенный преподавателем.
